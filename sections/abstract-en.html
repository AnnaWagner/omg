Abstract

The Ontology for Managing Geometry (OMG) is an ontology for describing geometry descriptions of objects. It provides means to support the application of multiple geometry descriptions of the same object as well as the description of the geometry evolution over time.

The namespace for OMG terms is http://www.w3id.org/omg#

The suggested prefix for the OMG namespace is omg

The Turtle version of the OMG ontology is available at http://www.w3id.org/omg/omg.ttl
The RDF/XML version of the OMG ontology is available at http://www.w3id.org/omg/omg.rdf
